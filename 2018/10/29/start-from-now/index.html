start from now

Posted on 2018-10-29

今天开始自己职业生涯中的第一篇博客，从无到有。

不想在各种大型博客网站混入其中，写博客的目的就是想把学到的知识再次巩固与记忆，今后方便查找。之前一直都是用word和onenote等软件做笔记的，虽然该记的内容都记下来了，需要用到的时候也能较为方便的查找，但这个功能仅限于需要回忆时才会翻开笔记查看之前写的内容；而多数的笔记是出于一种放置的状态的，并没有起到能及时复习的实质性作用。

所以我打算从今天起开始写自己的博客。

写博客和记笔记的区别在于：笔记是写给自己看的，博客是给所有人看的；所以在写博客时就会强迫自己更注意措辞，以一种更能让人理解的方式表达，同时，自己也需要真正的吃透一个知识点才能给别人讲清楚，所以在提升内容质量的同时还能训练集资的文字表达能力；在日后的学习与复习中，也能更快的回忆起之前的学习成果，何乐而不为呢？

So, why not start from now?
